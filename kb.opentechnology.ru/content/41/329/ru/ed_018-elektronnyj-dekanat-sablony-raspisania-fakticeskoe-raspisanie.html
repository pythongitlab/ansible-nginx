       6.1. Создание расписания для учебной группы

    Шаблон расписания — клише, представляющее собой условный список занятий по учебному процессу, расписанный на учебную неделю. В шаблон учебной недели записываются планируемые на учебную неделю занятия с указанием времени начала и длительности каждого занятия. При этом занятия могут указываться отдельно для четной или нечетной недели («мигающие» занятия), либо для четной и нечетной одновременно. Таким образом, шаблон расписания устанавливается на 2 условные недели — четную и нечетную. При создании шаблона расписания сами занятия не создаются (шаблоны не участвуют в учебном процессе), так как это всего лишь шаблон, слепок для создания реального расписания.

    Расписание — скопированный с шаблона расписания перечень занятий (при этом, это — уже действующие занятия, которые отражаются в учебном процессе).
    Создание штатного расписания для учебного процесса подразумевает собой создание шаблона и создание расписания из шаблона.

    1. Шаблоны расписания. Просмотр шаблонов расписания

    Для перехода к шаблонам расписания необходимо кликнуть по ссылке «Шаблоны расписания», расположенной на главной странице ЭД (в нужном подразделении):

    Рис. 1.1. Ссылка «Шаблоны расписания».

    В форме нужно указать Период и Режим отображения, выбрать форму занятия («Все», «Только дистанционно», «Только очно»). В системе предусмотрено четыре режима отображения:

    • По времени.
    • По учащимся.
    • По преподавателям.
    • По номеру кабинета.

    После выбора данных для фильтра необходимо нажать кнопку «Показать».

    С помощью панелей вкладок осуществляется фильтрация списка занятий, показываемых в шаблоне. Фильтры:

    • День недели (1-7).
    • Четность/нечетность недели (или Еженедельно).
    • Временной интервал (будут показаны шаблоны занятий в заданном временном интервале, либо за весь день, в случае, если выбрана опция «Все»).

    Фильтрация производится с учетом подразделения, в котором в данный момент находится пользователь.

    Рис. 1.2. Фильтрация с учетом подразделения.

    В результате фильтрации снизу будет отображен шаблон расписания, который выглядит как список шаблонов занятий, подходящий по параметрам фильтрации для учебных процессов заданного периода.

    2. Создание нового шаблона расписания

    Для создания нового шаблона расписания необходимо воспользоваться контекстным меню, открывающимся при клике на ссылку «Действия». Из меню нужно перейти по ссылке «Новый шаблон» (Рис. 2.1).

    Рис. 2.1. Контекстное меню для создания нового шаблона расписания .

    Откроется форма для создания нового шаблона:

    Рис. 2.2. Форма для создания нового шаблона.
    • День недели. День недели в котором будут проводится занятия, соответствующие шаблону (поз. 1-7 Рис. 2.3).
    • Учебная неделя. Возможно выбрать «Четная», «Нечетная» или «Еженедельно»(аналогия с «мигающими» занятиями в реальном учебном заведении).
    • Начало занятия, Длительность занятия, Форма занятия, Тип занятия, Подразделение, Кабинет. Эти поля интуитивно понятны для заполнения.
    • Учебный процесс. Учебный процесс, к которому привязан шаблон занятия (сами занятия тоже привязываются к учебному процессу)

    После нажатия кнопки «Сохранить», шаблон занятия для учебного процесса будет создан. Отобразится страница с данными только что созданного шаблона, отчет о нагруженности шаблона, пересечениях учебного процесса по учащимся и преподавателям (Рис. 2.3).

    Рис. 2.3. Страница с данными только что созданного шаблона.

    На данной странице:
    1. Дополнительное меню.
    2. Предупреждение о неправильной нагруженности учебного процесса (пересечения шаблонов).
    3. Информация о созданном шаблоне.
    4. Текущий статус шаблона и опция смены статуса; информационное сообщение: «Пересекаемых шаблонов не нашлось» (в случае отсутствия пересечений).
    5. Отчет о просматриваемом шаблоне.
    6. Отчет «Пересечение с шаблонами по ученикам».
    7. Отчет «Пересечение с шаблонами по учителям».

    2.1. Дополнительное меню

    (поз. 1. Рис. 2.3) Содержит пункты:
    Вернуться к списку шаблонов расписания.
    Вернуться в учебный план.
    Новый шаблон. Создаст новый шаблон (см. п. 2).
    Новый шаблон для учебного процесса. Создаст новый шаблон для того же учебного процесса, что и в просматриваемом шаблоне. 

    2.2. Нагруженность шаблона расписания

    Если нагруженность шаблона не соответствует нагруженности учебного процесса, для которого он создан, то над таблицей появится надпись «Неправильная нагруженность учебного процесса» (см. поз. 2 Рис.2.3).

    Нагруженность учебного процесса — суммарное количество действующих шаблонов занятий для данного учебного процесса. Содержит данные: сумма Часов в неделю, сумма Часов очно, сумма Часов дистанционно).
    Текущая нагруженность учебного процесса выделена в таблице красным цветом. Черным цветом выделены часы, прописанные в настройках учебного процесса. Корректным будет являться равенство часов нагруженности учебного процесса и значений, прописанных в настройках учебного процесса.

    Рис. 2.2.1. Неправильная нагруженность учебного процесса.

    При корректной нагруженности (8/8, 6/6, 2/2) данное предупреждение пропадет (не будет отображаться).

    Возможно появление других предупреждающих надписей.

    В случае отсутствия пересечений с другими шаблонами (см. п. 2.4. и п. 2.5), под кнопкой смены статуса выводится надпись: «Пересекаемых шаблонов не нашлось».

    3. Отчет о нагрузке шаблонов

    Со страницы списка шаблонов расписания для перехода к «Отчету о нагрузке шаблонов» необходимо:
    1) Воспользоваться контекстным меню, открывающимся при клике на ссылку «Действия».
    2) Из меню перейти по ссылке «Массовые действия»: 

    Рис. 3.1. Данные для отчета.

    Укажите Период, из которого будут взяты данные для отчета и тип отчета:

    • Нагруженность учебных процессов. Отчет о нагруженности всех учебных процессов данного периода в заданном подразделении (в этом случае необходимо указать Подразделение учебного процесса).
      Рис. 3.2. Отчет о нагруженности всех учебных процессов.
    • Пересечение шаблонов (по всем подразделениям). Отчет о всех пересечениях шаблонов по всем подразделениям.

    4. Массовые действия

    Интерфейс позволяет проводить массовое изменение времени начала занятий в шаблонах расписания.

    Со страницы списка шаблонов расписания для перехода к интерфейсу «Массовых действий» необходимо:
    1) Воспользоваться контекстным меню, открывающимся при клике на ссылку «Действия».
    2) Перейти по ссылке «Массовые действия»: 

    Рис. 4. Массовые действия.

    Здесь необходимо заполнить сетку изменений времени начала занятий и отправить форму. Система в порядке очередности задач изменит время во всех шаблонах, найденных в периоде с заданным значением времени начала занятия.

    5. Создание расписания

    После того как шаблон создан, можно создавать расписание. Для этого предусмотрена ссылка «Создать расписание» на странице шаблонов расписания. Создание расписания предполагает создание занятий в соответствии с созданными шаблонами занятий в шаблоне расписания в заданном периоде.
    Устаревшие функции. Функционал перенесен в «Календарь».

    Расписание можно создать либо для учебной недели (кнопка Создать учебную неделю), либо для учебного дня(кнопка Создать учебный день)

    В обоих случаях указываются следующие параметры:

    Период — период, в котором будет создано расписание.
    Выберите дату — дата, в которой будет создано расписание(в случае если создается учебный день) или дата начала учебной недели(в случае если создается учебная неделя).
    Учебная неделя — выбрать, четной или нечетной является неделя в которой создается расписание
    День недели — выбрать день недели.
    Обновить расписание — помечается в случае если на данную дату ранее уже было назначено расписание и необходимо его обновить.

    6. Решения

    6.1. Создание расписания для учебной группы

    Задача:
    Методисту ГОУ ВПО НИИ ЧАВО необходимо подготовить расписание занятий учебного процесса «Секреты мастерства» для группы из пяти студентов.
    Программа обучения состоит из одного элемента учебного плана (дисциплины), рассчитана на 12 академических часов, которые включают в себя девять дистанционных занятий по 1 академическому часу и одно очное практическое занятие длительностью 3 академических часа.
    Учебный процесс должен начаться 22.06.2021 года, начало занятий в 13-00, по рабочим дням.
    Общие предусловия:
    ГОУ ВПО НИИ ЧАВО — головное подразделение института, учебный процесс будет осуществляться непосредственно в нем.
    Учебный процесс «Секреты мастерства» проводится в рамках действующего учебного периода «Факультативное лето».
    Учебный процесс «Секреты мастерства» основывается на программе «Горячий факультатив».
    Программа «Горячий факультатив» состоит из одного элемента учебного плана (дисциплины) «Секреты мастерства».

    Алгоритм решения:

    1) Создание шаблона расписания.
    2) «Привязка» шаблона расписания к Календарю.

    1) Создание шаблона расписания.

    В сводной таблице учебных процессов (объект «Учебные процессы» ЭД) методист выбирает учебный процесс «Секреты мастерства» и, нажав на иконку «Создать шаблон расписания для данного учебного процесса», расположенную в столбце «Действия» (Рис. 6.1.10), переходит на страницу создания шаблона расписания.

    Рис. 6.1.10. Переход к созданию шаблона расписания учебного процесса «Секреты мастерства 2021».

     На странице шаблона расписания последовательно, без привязки к календарному графику, заполняет шаблоны расписания для всех рабочих дней недели (понедельник, вторник, среда, четверг, пятница), устанавливая (Рис. 6.1.11):
    «Начало занятия» — 13-00;
    «Длительность занятия» — 45 минут;
    «Форма занятия» — дистанционно;
    «Тип занятия» — обычный урок.

    Рис. 6.1.11. Создание шаблона расписания.

    После этого сформированный список шаблонов расписания будет доступен на странице «Список шаблонов на учебную неделю», перейти на которую можно из перечня учебных процессов, нажав на иконку «Просмотр списка шаблонов на неделю для учебного процесса» в столбце «Действия» учебного процесса «Секреты мастерства» (Рис. 6.1.12).

    Рис. 6.1.12. Недельный список шаблонов расписания учебного процесса.
    Подробная информация о шаблонах расписания в разделах статьи Шаблоны расписания. Просмотр шаблонов расписания и Создание нового шаблона расписания.

    2) «Привязка» шаблона расписания к Календарю.

     Несмотря на то, что в Электронном деканате предусмотрена возможность создавать занятия сразу на весь период («Автоматическое обновление/создание занятий на весь период»), рекомендуем применять именно инструмент «Создать занятия на неделю».
    В случае, если расписание создается на длительный период и по нему могут быть изменения или выявлены ошибки - последующее редактирование всего периода будет чересчур трудозатратным.

    В сводной таблице учебных периодов (объект «Учебные периоды» ЭД) методист выбирает учебный период «Факультативное лето» и, нажав на иконку «Просмотр календаря учебных дней», расположенную в столбце «Действия» (Рис. 6.1.13), переходит к календарю учебного периода.

    Рис. 6.1.13. Переход к календарю учебного периода.
    Обратите внимание: корректное выполнение последующих действий будет возможно при условии, что у вас выбрано конкретное подразделение и определены дни (рабочие, выходные, праздничные) на учебный период. О создании/заполнении дней на учебный период смотрите в статье «ЭД-019. Электронный деканат. Календарь. Фактическое расписание».

    На странице Календаря методист последовательно, нажимая на  кнопку   «Создать занятия на неделю», создает недельные расписания на календарный период, учитывающий продолжительность учебного процесса «Секреты мастерства» (Рис. 6.1.14).

    Рис. 6.1.14. Создание расписания на неделю.

    В результате шаблоны расписания учебного процесса «Секреты мастерства 2021» будут привязаны к конкретным неделям учебного периода «Факультативное лето» (Рис. 6.1.15).

    Рис. 6.1.15. Расписание учебного процесса «Секреты мастерства 2021», привязанное к датам Календаря.

    В процессе создания недельных расписаний система сформировала сообщения об ошибках (Рис. 6.1.16).

    Рис. 6.1.16. Сообщения об ошибках в процессе формирования недельных расписаний.

    Это связано с тем, что даты начала и окончания учебного процесса не совпадают с началом и окончанием календарной недели  и шаблоны расписания оказались привязаны к «лишним» дням.
    Последовательно переходя на страницы календарных дней, отмеченных иконкой , методист производит удаление занятий на эти дни (Рис. 6.1.17).

    Рис. 6.1.17. Удаление расписания занятий на выбранный в Календаре день.

    Расписание создано (Рис. 6.1.18).

    Рис. 6.1.18. Расписание учебного процесса «Секреты мастерства 2021» в рамках учебного периода «Факультативное лето 2121».

    Теперь в Журнале группы (Рис. 6.1.19) к календарным датам привязаны события расписания и контрольные точки учебно-тематического плана .

    Рис. 6.1.19. Журнал группы Группа01.

    В связи с тем, что последнее занятие программы отличается от заданного в шаблоне по длительности, методист корректирует в Журнале группы его параметры (Рис. 6.1.20).

    Рис. 6.1.20. Корректировка длительности занятия.
